# Colors Wars

Projets which create 1v1 PVP game in opengl 

## Git
https://github.com/Gruequotient3/Colors_War.git

## Building 

### Unix

```bash
git clone https://github.com/Gruequotient3/Colors_War.git
cd Colors_War/
git submodule update --init --recursive 
make libs && make
```

## Using

```bash
make
./main
```
